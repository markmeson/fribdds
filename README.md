# fribdds
Cross Platform DDS Image Viewer


Requires: 

- wxWidgets 3.0


Compiling on Linux:

 - make
 - ./install.sh


Compiling on Windows

 - not sure

Supports:

- 32 bit color

- 16 bit color (5:6:5, 4:4:4:4, and 1:5:5:5)

- DXT1 Compressed Images


Does Not Support:
- Non-DXT1 Compressed DDS Images
